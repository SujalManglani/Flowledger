💰 Flowledger – Smart Finance Dashboard

A modern finance dashboard built using Vue.js with role-based access, interactive charts, and a clean glassmorphism UI.

---

🚀 Features

- 🔐 Role-Based Access
  
  - Admin: Add & delete transactions
  - Viewer: Read-only access

- 📊 Interactive Charts
  
  - Category breakdown (Pie Chart)
  - Transaction trends (Wave Chart)

- 💸 Transaction Management
  
  - Add new transactions (Admin only)
  - Delete transactions
  - Live transaction list

- ✨ Modern UI/UX
  
  - Glassmorphism design
  - Blur + glow effects
  - Smooth animations
  - Parallax background

- 📈 Live Stats
  
  - Total amount counter (animated)
  - Transaction count
  - Status display

---

🛠️ Tech Stack

- Vue 3 (Composition API)
- Vite
- Chart.js
- CSS (Glassmorphism + Animations)

---
🔑 How to Use
Select role from sidebar:
Admin
Viewer
As Admin:
Add transactions using the form
Delete transactions from list
As Viewer:
Only view data (no changes allowed)
